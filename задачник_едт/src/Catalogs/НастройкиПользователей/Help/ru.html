Предназначен для ведения внешних контрагентов.

Справочник можно открыть соответствующей командой в разделе Сервис.

В справочник вводится:

  • Наименование настройки;
  • Настройки по обращениям;
    • Автооткрытие формы обращения - запуск формы обращение при старте;
    • Тип применяемого обращения - есть выбор нескольких параметров:
      • Только внутренние обращения -  при создании нового обращения выбор будет только по отделам;
      • Только внешние обращения -  при создании нового обращения выбор будет только по контрагентам;
      • Внутренние и внешние обращения - при создании нового обращения можно выбрать вид обращения;
    • Обращения по умолчанию - при создании нового обращения по умолчанию будет применятся
      "Внутренний" или "Внешний" вид обращения.
