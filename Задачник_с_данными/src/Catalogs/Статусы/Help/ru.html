Предназначен для ведения статусов обращений.

Справочник можно открыть соответствующей командой в разделе Справочники.

Справочник имеет несколько предопределенных элеменов - "Новое", "Выполнено", "Закрыто".

В справочник вводится:

  • Наименование статуса;
  • Цвет строки - таким цветом будет отображаться строка в списке обращений имеющая данный статус;
  • Процент выполнения - устанавливается процент выполнения по текуему статусу;
  • Количество часов выполнения - устанавливается количество часов за которое работы по данному статусу должны быть выполнены;
  • Цвет строки при превышении часов выполнения - таким цветом будет отображаться строка в списке обращений превышающая
    количество часов выполнения текущего статуса.
