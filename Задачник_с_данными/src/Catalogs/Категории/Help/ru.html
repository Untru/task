Предназначен для ведения категорирования проблем.

Справочник можно открыть соответствующей командой в разделе Справочники.

Справочник имеет несколько уровней вложенности, тем самым можно сгруппировать схожие проблемы.

В справочник вводится:

  • Наименование категории;

Для автоматической подстановки в обращение:

  • Заголовок обращения;
  • Отдел, в который будет адресовано обращение;
  • Основной исполнитель, по выбранному отделу.
